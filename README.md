# Telco Customer Analytics Notebook

Run instructions:
1. Install dependencies:
   `pip install -r requirements.txt`
2. Configure Kaggle credentials:
   - Place `kaggle.json` at `~/.kaggle/kaggle.json`, or
   - Set `KAGGLE_USERNAME` and `KAGGLE_KEY` env vars.
3. Open and run:
   `notebooks/telco_customer_analytics.ipynb`

Outputs (figures/tables) can optionally be saved to `outputs/`.
